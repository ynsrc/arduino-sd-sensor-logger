# YNSRC - SD Sensor Logger - Arduino Project

# Parts
1. Arduino UNO R3
2. Rotary Encoder
3. DS3231 RTC Module
4. DHT11 Humidiy and Temperature Sensor
5. SD Card Module
6. SD Card
7. 16x2 LCD Display with I2C Interface
